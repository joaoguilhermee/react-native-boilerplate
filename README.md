# react-native-boilerplate

Initial react native projects.

I've built with some features for examples.

## Run fake server
```
 json-server --watch db.json -p 3333 
 ```
## Install project
```
yarn install
```
## Run project 
```
react-native run-ios 
```
or
```
react-native run-android
```

## Run tests 
```
yarn test  --env=jsdom
```
or
```
react-native run-android
```

## This project is using
- styled-components
- react-navigation
- axios
- redux
- immer
